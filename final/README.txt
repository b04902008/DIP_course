This is the revised implementation of the domain transform filters described
in the paper, only used for my final project of the course:

   Domain Transform for Edge-Aware Image and Video Processing
   Eduardo S. L. Gastal  and  Manuel M. Oliveira
   ACM Transactions on Graphics. Volume 30 (2011), Number 4.
   Proceedings of SIGGRAPH 2011, Article 69.

For example usage of my final project see the file:

   src/README.m


Please refer to the publication above if you use this software. For an
up-to-date version go to:

   http://inf.ufrgs.br/~eslgastal/DomainTransform/

The reference implementation downloaded from the website above is located
in the folder:

   DomainTransformFilters-Source-v1.0
